8)Mind your Ps and Qs(Cryptography-RSA): -

i)We have a file, which has c, n and e values
ii)Here, we need to decode the c, i.e, the cypher text, hence we use dcode's RSA tool
iii)We input the value of n and e to decode the c
OR
i)We use RSA formulas
ii)We need to factorise the n string into p and q
->n = pq (Prime factorization)
iii)From p and q, we get the λ.
->λ(n) = lcm(p − 1, q − 1). 
iv)Then, we calculate d that is the decryption key,
->d ≡ e^−1 (mod λ(n))
v)Using d, we find the value of c
->c=(d^c) mod n
vi)This gives us the de-cyphered text
